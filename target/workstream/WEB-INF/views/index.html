Protocube

Protocube is a puzzle game that is not like Tetris at all! It was written completely in Javascript/CSS/HTML with a tiny bit of java for score keeping. I provided all of the scripting while all of the graphics (and most of the CSS/HTML) were produced by Ozone

Panoramic Map

Using the Google Maps API, I've added markers for every place that I've taken a panoramic. Click each marker to see the panoramic. Zoom in for exact locations.
